* Development

- Use =when-let*= instead of =when-let=.
- Show date or series number in section title.

* Version 1.2 (2025-11-15)

- ~dicom-next~, ~dicom-previous~: New commands to move to next or previous image.
- ~dicom-thumb-height~: New option to configure thumbnail image pixel height.
- Use ~dcm2img~ instead of deprecated ~dcmj2pnm~.
- Improve handling of DICOM files with invalid encoding. Use Emacs encoding
  conversion instead of ~--convert-to-utf8~ from ~dcm2xml~.
- Improve image conversion of DICOM files by falling back to ImageMagick in case
  ~dcm2img~ fails.

* Version 1.1 (2025-10-13)

- Handle empty sequences in DICOM files.
